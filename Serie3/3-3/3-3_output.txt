3. Problem:

Wenn |int limit| kein Vielfaches von 3 ist, ist die Schleife endlos, da |counter != limit| immer |true| ist.

Das Problem tritt erst beim Ausführen auf, da der Compiler keine Parameter-Sanity-Checks bezogen auf den Methoden-Body durchführt.

Korrektur-Beispiel:

public static String dreierReihe(int limit) {
  String result = "";
  
  if (limit < 3) {
    return "Limit zu klein (< 3)";
  }
  
  for (int counter = 3; counter <= limit; counter += 3) {
    result = result + counter + " ";
  }
  return result;
}
